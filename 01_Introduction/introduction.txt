01. INTRODUCTION TO PYTHON

What is Python?
---------------
Python is a simple and easy-to-learn programming language. It helps us give instructions to a computer so it can perform tasks.

Python is one of the most popular programming languages because its syntax is easy to read and understand.

Python is used in:
- Web Development
- Artificial Intelligence (AI)
- Machine Learning (ML)
- Data Science
- Automation
- Game Development
- Software Development

Example:
print("Hello, World!")

Output:
Hello, World!


Who Developed Python?
---------------------
Python was created by Guido van Rossum, a programmer from the Netherlands.

He started working on Python in the late 1980s, and the first version was released in 1991.

Because he created Python, he is known as the "Father of Python".

Quick Facts:
- Creator: Guido van Rossum
- First Release: 1991
- Language Type: High-Level Programming Language
- Maintained By: Python Software Foundation (PSF)


Features of Python
------------------

1. Easy to Learn
Python uses simple English-like syntax, making it beginner-friendly.

2. Easy to Read
Python code is clean and easy to understand.

3. Free and Open Source
Anyone can download and use Python for free.

4. Platform Independent
Python programs can run on Windows, macOS, and Linux.

5. Large Community Support
Millions of developers use Python and share helpful resources.

6. Versatile Language
Python can be used in many fields such as AI, web development, automation, and data science.


Installing Python
-----------------

Step 1: Download Python
Visit the official website:
https://www.python.org/downloads/

Step 2: Install Python
- Download the latest version.
- Run the installer.
- Select "Add Python to PATH".
- Click "Install Now".

Step 3: Check Installation

Command:
python --version

Example Output:
Python 3.13.5


Running Python Programs
-----------------------

Method 1: Python Interpreter

Open Terminal or Command Prompt and type:

python

Then write:

print("Hello Python")

Output:
Hello Python


Method 2: Using VS Code

Create a file named:
hello.py

Write:

print("Welcome to Python!")

Run:

python hello.py

Output:
Welcome to Python!
