02. VARIABLES AND DATA TYPES

Variables
---------
A variable is like a box that stores information.

We use variables to store data such as names, ages, marks, and prices.

Example:

name = "Deepu"
age = 21

print(name)
print(age)

Output:
Deepu
21


Rules for Variable Names
------------------------
1. A variable name can contain letters, numbers, and underscores (_).
2. A variable name cannot start with a number.
3. Spaces are not allowed.
4. Variable names are case-sensitive.

Examples:

student_name = "Rahul"   ✓
age = 20                ✓

2name = "Rahul"         ✗
student name = "Rahul"  ✗


Data Types
----------
A data type tells Python what kind of data is stored in a variable.

The most common data types are:

1. Integer (int)
2. Float (float)
3. String (str)
4. Boolean (bool)


1. Integer (int)
----------------
Integers are whole numbers without decimal points.

Examples:

age = 21
marks = 95

print(age)
print(marks)

Output:
21
95


2. Float (float)
----------------
Float values are numbers that contain decimal points.

Examples:

price = 99.99
height = 5.8

print(price)
print(height)

Output:
99.99
5.8


3. String (str)
---------------
A string is text written inside single quotes (' ') or double quotes (" ").

Examples:

name = "Deepu"
city = "Hyderabad"

print(name)
print(city)

Output:
Deepu
Hyderabad


4. Boolean (bool)
-----------------
A Boolean value can have only two values:

True
False

Examples:

is_student = True
is_working = False

print(is_student)
print(is_working)

Output:
True
False


Checking Data Types
-------------------

We can use the type() function to check the data type of a variable.

Example:

name = "Deepu"
age = 21

print(type(name))
print(type(age))

Output:

<class 'str'>
<class 'int'>


Type Conversion
---------------
Type conversion means changing one data type into another.

Example:

a = "10"
b = int(a)

print(b)
print(type(b))

Output:
10
<class 'int'>


More Examples:

num = 25
print(float(num))

Output:
25.0


price = 99.9
print(int(price))

Output:
99


Input in Python
---------------
The input() function is used to take input from the user.

Example:

name = input("Enter your name: ")

print("Hello", name)

Output:

Enter your name: Deepu
Hello Deepu


Taking Number Input
-------------------

age = int(input("Enter your age: "))

print(age)

Output:

Enter your age: 21
21


Output in Python
----------------
The print() function is used to display information on the screen.

Example:

print("Welcome to Python")

Output:
Welcome to Python


name = "Deepu"

print("My name is", name)

Output:
My name is Deepu


Key Points
----------

✓ Variables store data.
✓ Data types tell Python what kind of data is stored.
✓ Integer stores whole numbers.
✓ Float stores decimal numbers.
✓ String stores text.
✓ Boolean stores True or False.
✓ input() is used to take input from the user.
✓ print() is used to display output.
✓ type() is used to check data types.

Interview Questions
-------------------

Q1. What is a variable?

A variable is a container used to store data in memory.

Q2. What are data types?

Data types define the type of value stored in a variable.

Q3. Name the four basic data types in Python.

Integer, Float, String, and Boolean.

Q4. What is type conversion?

Type conversion is the process of changing one data type into another.

Q5. What is the difference between input() and print()?

input() is used to take input from the user, while print() is used to display output on the screen.
